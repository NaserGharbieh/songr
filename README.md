# Songr Spring MVC app 
## This is a project for applaying what I learn in Spring MVC
## Curuntly this project has 5 routes:
- ### /hello 
- That of course returns a html page with the text : "hello world ".
- ### /capitalize/smalltext
- Which returns anhtml page with any text in UPPERCASE.
- ### /albums
- Which returns a html page with a **Form to add any album you want**.
- ### /songs
- Which returns a html page with a **Form to add any song you want**.
- ### /album/{albumId}
- Which returns a html page with  **the album and the songs added to it**.
