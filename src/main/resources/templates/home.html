Songr Spring MVC App

This is a project for applying what I learned in Spring MVC.

Available Routes:

  • /hello - Returns an HTML page with the text: "hello world"
  • /capitalize/smalltext - Returns an HTML page with any text in UPPERCASE.
  • /albums - Which returns a html page with a Form to add any album you want.

  • /songs - Which returns a html page with a Form to add any Song you want.
  • /album/{albumId} - Which returns a html page with the album and the songs added to it
